Spring Framework

Messaging, AOP, Transactional Data and Persistence, Spring MVC Web Framework, Spring WebFlux Reactive Web Framework, ...

Ioc, Inversion of Control Container

Beans, Context, Core, SpEL

The process of an object itself controlling the instantiation or location of its dependencies(objects it works with) is inverted. The control is moved away from this object to the IoC container.

The Spring Framework uses a technique called dependency injection(DI) to implement the idea of IoC

Spring boot

The Spring Framework requires loads of configuration to get started.

Spring Boot extends the Spring Framework and provides a quick way to create a production-ready Spring-based application.

start.spring.io ; 에서 프로젝트 생성

build; maven으로 함. java version; 17, 필요한 dependency 설치(일단 security 제외)

프로젝트 기본 구조

src 밑에 main과 test가 있음

resources 밑에 html(js, css 포함)을 넣으면 rest가 아닌 프로젝트를 생성하면 여기서 파일을 불러와서 적용 가능

이때 web template을 설치해야함(thymleaf)

application.properties(or yml)에 프로젝트 수행에 관련된 설정을 함(내용이 많음)
